# LED Chaser

A simple Arduino project that creates a running light effect using multiple LEDs.

## Components Used
- Arduino Uno
- 5 LEDs
- 5 × 220Ω resistors
- Breadboard
- Jumper wires

## Working
The LEDs turn ON and OFF one after another, creating a running or chasing light effect.
